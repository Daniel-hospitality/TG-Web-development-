CharAt

De methode charAt () retourneert het teken op de opgegeven index in een tekenreeks. De index van het eerste teken is 0, het tweede teken is 1, enzovoort.

Tip: De index van het laatste teken in een string is string .length-1, het voorlaatste karakter is string .length-2, enzovoort,
https://www.w3schools.com/jsref/tryit.asp?filename=tryjsref_charat

var str = "HELLO WORLD";
var res = str.charAt(0)
Antw = H

var str = "HELLO WORLD";
var res = str.charAt(5)
Antw = O

Concat()

De concat () methode wordt gebruikt om twee of meer strings samen te voegen. Deze methode verandert de bestaande strings niet, maar retourneert een nieuwe string met de tekst van de samengevoegde strings.
https://www.w3schools.com/jsref/tryit.asp?filename=tryjsref_concat_string

var str1 = "Hello ";
var str2 = "world!";
var res = str1.concat(str2);

slice()

De methode slice () extraheert delen van een string en retourneert de geëxtraheerde delen in een nieuwe string. Gebruik de begin- en eindparameters om het deel van de tekenreeks op te geven dat u wilt extraheren. Het eerste teken heeft de positie 0, de tweede heeft positie 1, enzovoort. Tip: gebruik een negatief getal om te selecteren vanaf het einde van de string.

var str = "Hello world!";
var res = str.slice(0, 5);
Antwoord is hello h=0 & o=5

split

De methode split () wordt gebruikt om een ​​tekenreeks op te splitsen in een reeks subtekenreeksen en retourneert de nieuwe reeks. Tip: als een lege tekenreeks ("") wordt gebruikt als scheidingsteken, wordt de tekenreeks over elk teken verdeeld. Opmerking: de methode split () verandert de oorspronkelijke tekenreeks niet.

var str = "How are you doing today?";
var res = str.split(" ");
Antw= how,are,you,doing,today

substr()

De methode substr () extraheert delen van een tekenreeks, beginnend bij het teken op de opgegeven positie, en retourneert het opgegeven aantal tekens.

var str = "Hello world!";
var res = str.substr(1, 4);
Atw= ello

var str = "Hello world!";
var res = str.substr(2);
Ant= llo world!
