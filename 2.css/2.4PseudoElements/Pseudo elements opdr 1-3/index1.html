Antwoord opdracht 1

  • ::after (:after) Creëert in CSS een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de contenteigenschap. Het is standaard inline.
  • ::before (:before) Creëert in CSS een pseudo-element dat het eerste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de contenteigenschap. Het is standaard inline.
  • ::first-line Kan bijvoorbeeld worden gebruikt om het lettertype of de kleur ect, van de eerste regel van een alinea te wijzigen.
  • ::first-letter kan gebruikt worden om de eerste letter aan te passen
