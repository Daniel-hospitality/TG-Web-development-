Antwoord opdracht 2

  • ::after (:after) Creëert in CSS een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de contenteigenschap. Het is standaard inline.bv een gifje achter het einde van een Paragraph.
  • ::before (:before) Creëert in CSS een pseudo-element dat het eerste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de contenteigenschap. Het is standaard inline. bv een gifje voor de Paragraph.
  • ::first-line Kan bijvoorbeeld worden gebruikt om het lettertype of de kleur ect, van de eerste regel van een alinea te wijzigen.
  • ::first-letter kan gebruikt worden om de eerste letter aan te passen
  • backdrop Het backdrop pseudo-element is een box waar het full screen element achter zit, maar tegelijkertijd boven alle andere content staat. Het kan samen met de: fullscreen pseudo-class of het (dialog) HTML5-element worden gebruikt om de achtergrondkleur van het gemaximaliseerde scherm te wijzigen, voor het geval we niet voor het standaard zwart willen kiezen.
  • cue Is vergelijkbaar met WebVTT, wordt gebruikt voor bv (Getimde) ondertiteling en of bij schriften.
  • cue-region is het zelfde als bovenstaande maar dan voor het stylen van bijschriften of ondertitels.
  • grammar-error wordt gebruikt om taal fouten een style mee tegen vb doorstreept of kleur,
  • marker Vb li lijstitem aanpassen en of veranderen inplaat van stip of nummer, vb item 1, item 2, dus selecteert het markeringsvak van een lijstitem, dat doorgaans een opsommingsteken of nummer bevat.
  • part vertegenwoordigt elk element binnen een schaduwboom met een overeenkomend attribuut.
  • placeholder Het pseudo-element verwijst naar de tijdelijke aanduiding van invoerelementen in uw presentatie, meestal te vinden in formulieren. De meeste browsers passen standaard een lichtgrijze kleur toe op tijdelijke aanduidingen.
  • selection Bij het selecteren van tekst kan je vb tekst kleur en of achtergrond kleur veranderen.
  • slotted vertegenwoordigt elk element dat in een sleuf in een HTML-sjabloon is geplaatst
  • spelling-error represents a text segment which the user agent has flagged as incorrectly spelled.
