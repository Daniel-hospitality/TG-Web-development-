Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld;

  • type selectors
  • descendant selectors
  • class selectors

Antwoord 3

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • Type selectors

    • vb <H1>, <p>, <spam>, <div>

    Een Type Selector (ook wel Element Type Selector genoemd) koppelt elementen aan de corresponderende elementknooppuntnaam

    p { /* "p" is the type selector */ margin: 0 0 1em 0; }

  • Descendant selectors

    • = vb ul li { } header h2 { } footer a { } .module div { } #info-toggle span { } div dl dt a { }

    A descendant selector in CSS is any selector with white space between two selectors without a combinator. Here’s some examples:

    Take ul li { } for example. It means “any list item that is a descendant of an unordered list.”

    To illustrate, div span { } will match:

    <div>
    <span>I will match</span>
    <ul>
    <li> <span>I will match too</span>
    </li>
    </ul>
    </div>

  • Class selectors

    • A class selector in CSS starts with a dot <.>, like this:

    .class { }

    vb <button class="big-button">Push Me </button>

    is selected and styled like this:

    .big-button { font-size: 60px; }
